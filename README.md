# Photoshop HP Copyright PSD

Photoshopで開いている作品PSDから、HP掲載用のコピーPSDを作成し、固定テンプレートPSD内のコピーライト用グループを右下に自動配置するExtendScriptです。

## v1でできること

- 現在アクティブな保存済みPSDを対象にします。
- 元PSDを直接変更せず、同じフォルダに `*_HP.psd` を作成します。
- テンプレートPSDから `クレジット白` グループだけを複製します。
- 複製したグループを最前面へ配置します。
- 画像の縦横比に応じてグループをリサイズします。
- 右4%・下4%の安全マージン線に接するよう右下へ配置します。
- 配置後の `_HP.psd` を保存します。

JPG書き出し、画像リサイズ、解像度変更、ガイド作成、フォルダ一括処理はv1ではまだ自動化していません。

## 実行方法

macOSのPhotoshopで対象PSDを開いた状態で、以下から実行します。

```text
ファイル > スクリプト > 参照...
```

その後、`scripts/add_hp_copyright.jsx` を選択してください。

対象PSDに未保存変更がある場合は処理を中止します。先に元PSDを保存してから実行してください。

## 初回設定

初回実行時、または保存済み設定ファイルがない場合は、クレジット用テンプレートPSDの選択ダイアログが表示されます。

選択したテンプレートPSDのパスは、Photoshop/ExtendScriptのユーザー領域に保存され、次回以降は自動で読み込まれます。

テンプレートPSDには、名前が完全一致する以下のレイヤーグループが必要です。

```text
クレジット白
```

このグループが見つからない場合、処理は中止されます。

## 出力PSD

元ファイルが `sample.psd` の場合、同じフォルダに以下を作成します。

```text
sample_HP.psd
```

すでに `sample_HP.psd` が存在する場合は、次のいずれかを選択できます。

- 上書きする
- 別名で保存する
- キャンセル

上書き対象の `_HP.psd` がPhotoshopで開かれている場合は、事故防止のため処理を中止します。閉じてから再実行してください。

別名で保存する場合は、`sample_HP_2.psd`、`sample_HP_3.psd` のように、存在しない番号を探して保存します。

## 補足

安定運用後は、Photoshopの `Presets/Scripts` に `scripts/add_hp_copyright.jsx` へのシンボリックリンクを置くと、スクリプトメニューから直接実行しやすくなります。
